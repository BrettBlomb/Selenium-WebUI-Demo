# Selenium WebUI Automation Demo

![Tests](https://github.com/BrettBlomb/Selenium-WebUI-Demo/actions/workflows/selenium-tests.yml/badge.svg)

Web UI automation framework using Python, Selenium, Pytest, and GitHub Actions CI.

## What This Project Demonstrates
- Headless Chrome automation (CI-friendly)
- DOM element lookup strategies
- Search workflow validation
- CI/CD pipeline with automated browser testing

## Tools & Technologies
- Python, Selenium, WebDriver Manager
- GitHub Actions
- Headless Chrome

## Run Tests
pip install -r requirements.txt
pytest -v
